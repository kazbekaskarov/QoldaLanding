Q01 На каких ОС работает?

Windows 10/11 (x64, arm64), macOS 13+ (Apple Silicon, Intel), Linux (Ubuntu 22.04+, Fedora 38+, X11 полностью, Wayland частично).

Q02 Это правда demo?

Да. v0.2, публичный preview. Будут баги. Stable-релиз — позже, дату не обещаем.

Q03 Нужен ли интернет?

Только при первой загрузке модели Whisper. Дальше — полностью offline.

Q04 Сколько весит модель?

large-v3-turbo ≈ 1.6 ГБ (по умолчанию), small — 488 МБ.

Q05 Безопасно ли давать Accessibility-права?

Да. Исходники открыты, проверяй сам. Только курсор и клавиатура, никакого экрана/микрофона в облако.

Q06 Можно ли в Zoom / созвонах?

Да. Скажи «стоп» — печать с голоса отключается, курсор продолжает работать. «слушай» — обратно.

Q07 Что если у меня дёргается голова?

Сглаживание EMA + dwell-click + precision mode по голосу.

Q08 Очки / борода / наушники мешают?

MediaPipe устойчив. Проверь сам в demo выше.

Q09 Wayland?

Курсор и клавиатура — да. Always-on-top бар — ограниченно. Работаем над этим.

Q10 Можно ли коммерчески использовать?

Да. MIT.
